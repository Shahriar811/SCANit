# SCANit

SCANit is an Android application that turns your device into a powerful portable scanner. It simplifies the process of digitizing documents by using your phone's camera to capture high-quality images and extract text.

## Features

*   **Document Scanning:** Easily scan documents, receipts, notes, and more using your device's camera.
*   **Text Recognition (OCR):** Advanced OCR technology to accurately extract text from your scanned documents.
*   **Export Options:** Save your scanned documents as PDF or other formats.
*   **User-Friendly Interface:** A clean and intuitive interface built with Jetpack Compose for a seamless user experience.

## Technologies Used

*   **[Kotlin](https://kotlinlang.org/)**: The primary programming language for building the application.
*   **[Jetpack Compose](https://developer.android.com/jetpack/compose)**: Android's modern toolkit for building native UI.
*   **[CameraX](https://developer.android.com/training/camerax)**: For easy and consistent camera functionality across Android devices.
*   **[ML Kit Text Recognition](https://developers.google.com/ml-kit/vision/text-recognition)**: To recognize and extract text from images.
*   **[iText7](https://itextpdf.com/products/itext-7)**: For creating and manipulating PDF documents.
*   **[Apache POI](https://poi.apache.org/)**: To work with Microsoft Office documents.
*   **[Material Design 3](https://m3.material.io/)**: The latest version of Google's design system.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Android Studio Iguana | 2023.2.1 or later
*   Android SDK 26 or later

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/Shahriar811/SCANit.git
    ```
2.  Open the project in Android Studio.
3.  Let Android Studio sync the project and download the required dependencies.
4.  Run the app on an Android device or emulator.
